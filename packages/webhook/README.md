# @indent/webhook

Install using NPM:

```
npm install @indent/webhook
```

Or with yarn:

```
yarn add @indent/webhook
```

## Usage

```bash
# Test with defaults
indent-webhook-test http://localhost:4040

# Use your own test configuration
indent-webhook-test -c config.json http://localhost:4040
```
